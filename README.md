# .github

This repo holds org-level community health files for [openagp](https://github.com/openagp):

- [`profile/README.md`](profile/README.md) — renders on the org landing page
- [`GOVERNANCE.md`](GOVERNANCE.md) — governance model and neutralization roadmap
- [`CONTRIBUTING.md`](CONTRIBUTING.md) — how to contribute (DCO sign-off model)
- [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) — Contributor Covenant 2.1
- `ISSUE_TEMPLATE/` — default issue templates *(coming)*

These files are picked up automatically by every repo in the org that doesn't override them locally.
